# Algoritmo Genético Padrão para o Problema do Caixeiro Viajante (PCV)

Este repositório contém o código fonte desenvolvido como parte do Trabalho de Conclusão de Curso (TCC) em Ciência da Computação na UNIPAMPA.

## Pré-requisitos

* **Java JDK 17** ou superior.
* Qualquer IDE Java de sua preferência (Eclipse, IntelliJ IDEA, NetBeans, VS Code).

## Como Executar

Este projeto foi desenvolvido para ser simples e direto, sem a necessidade de configurações complexas de gerenciadores de dependência.

Para rodar o algoritmo, siga os passos abaixo:

1.  **Baixe o projeto:**
    * Faça o clone deste repositório ou baixe o arquivo `.zip` e extraia em seu computador.
2.  **Importe na IDE:**
    * Abra sua IDE Java.
    * Selecione a opção de **"Open Project"** (Abrir Projeto) ou **"Import Project"** e aponte para a pasta descompactada.
3.  **Execute:**
    * Localize a classe principal (geralmente nomeada como `TSPGeneticAlgorithm.java` no pacote principal).
    * Clique com o botão direito e selecione **"Run"** (Executar).

O algoritmo iniciará a execução utilizando a instância configurada (padrão: `kroA100`) e exibirá os resultados no console.
